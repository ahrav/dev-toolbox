# Dev Toolbox

A dumping ground for CLI tools, AI resources, MCP servers, and other development utilities.

## Contents

### Claude Code Skills Archive

**File:** `claude-skills.zip` (6.9MB)

Complete archive of Claude Code skills from `~/.claude/skills`. This includes all custom skills, tools, and configurations.

#### Included Skills:

**Creative & Design:**
- algorithmic-art - Generative art using p5.js
- canvas-design - Visual art in PNG/PDF format
- slack-gif-creator - Animated GIFs for Slack

**Document Processing:**
- docx - Document creation and editing
- pdf - PDF manipulation toolkit
- pptx - Presentation creation and editing
- xlsx - Spreadsheet operations

**Development Tools:**
- artifacts-builder - React/Tailwind/shadcn artifacts
- cdk-typescript-reviewer - CDK code review
- mcp-builder - MCP server development

**Performance & Optimization:**
- rust-hotspot-finder - Performance analysis
- rust-perf-triage - Benchmark analysis

**Communication:**
- brand-guidelines - Anthropic brand styling
- internal-comms - Internal communications templates
- theme-factory - Artifact theming

**Meta Tools:**
- product-self-knowledge - Anthropic product reference
- skill-creator - Skill development guide

#### Extraction:

```bash
# Extract to your Claude skills directory
unzip claude-skills.zip -d ~/
```
